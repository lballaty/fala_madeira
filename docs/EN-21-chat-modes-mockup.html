EN-21 — Chat modes: before / after

Decision #4 = the "prompt-mode split." Verified against the actual system instruction (_shared/gemini.ts:111-150): today free chat runs one rigorous training-SIMULATION tutor — it's told to self-interrupt, switch scenario every 2–3 turns, simulate "Diz?" misunderstandings and escalate — and to format Portuguese as Português / Pronunciation / English "for translations." So a casual "can we chat?" or an app question gets pulled into that drill rather than a natural reply or an answer. After: an intent picker routes you, and each mode behaves correctly — conversation is natural, translation only when you ask, help is interpreted as an app question.
